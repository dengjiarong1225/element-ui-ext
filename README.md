# element-ui-ext

## Quick Start

在程序入口添加

```
import App from './App'
import store from './store'
import ElementExt from 'element-ui-ext'
```

调用

```
Vue.use(ElementExt, { store })
```

## About

秉着“少写html，多写json”的原则，在此根据业务需求对element-ui2.0的部分组件进行了二次封装，主要包括表格、表单项、一般表单、查询表单、弹窗等系列组件，尽量做到了数据的可配置化。更多详细的api请以element-ui官网为准：https://element.eleme.cn/ 。
